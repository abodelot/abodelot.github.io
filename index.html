Cosmoscroll c++sfmlGPLv3

CosmoScroll is a space-based shoot'em up game, written in C++/SFML.

Inspired by games such as R-Type and Gradius, it features side-scrolling, retro-looking graphics and chiptune music.

Wallbreaker c++sfmlGPLv3

Wallbreaker is a clone of the famous Arkanoid/Breakout games, written in C++ with SFML.

The game features various power-ups, 20 levels, and a level editor.

jQuery JSON Viewer javascriptjqueryMIT

jQuery JSON Viewer is a jQuery plugin that displays a JSON object in an HTML tree view.

Features:

  • Syntax highlighting
  • Collapse and expand child nodes on click
  • Clickable links
  • Easily readable and minimal DOM structure
  • Optional support for libraries supporting big numbers
